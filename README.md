# Villa Real Estate System

A Django-based real estate platform where builders can list properties and users can browse and send inquiries.

## Features
- Property listing system
- Admin approval workflow
- User inquiry management
- Secure user authentication

## Tech Stack
- Django
- HTML, CSS, Bootstrap
- SQLite / MySQL

## How to Run
1. Clone repository
2. Install dependencies
3. Run server:
   python manage.py runserver
